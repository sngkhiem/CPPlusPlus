# CP++
A DSL for Competitive Programming Test generation, and Solution Verification.

## Environment Setup

### Prerequisites
Java compiler.

### Installation Steps


1. **Clone the repository**
   ```bash
   git clone https://github.com/sngkhiem/CPPlusPlus
   cd CPPlusPlus

2. **Create the environment from PPl.yml file:** 
    ```bash
    conda env create -f PPL.yml

3. **Activate the environment:**
    ```bash
    conda activate PPL
